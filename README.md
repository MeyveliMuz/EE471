# Week 5: Vibe Coding ve IDE Karşılaştırmaları

Bu hafta kapsamında yapay zeka destekli kodlama (vibe coding) araçlarını deneyimleme fırsatı buldum. Bu konudaki kişisel değerlendirmelerim aşağıdadır:

## Vibe Coding IDE Tercihi
Proje geliştirme sürecimde ana geliştirme ortamı (IDE) olarak **Antigravity**'i tercih ettim. Sağladığı akıcı geliştirme deneyimi ve doğal etkileşim biçimi, kodlama süreçlerimi oldukça kolaylaştırdı.

## Araçların "Clean Code" Deneyimleri (conform.py)
Çalışmalarım kapsamında `conform.py` dosyasını temel alarak farklı araçların Clean Code (Temiz Kod) standartlarında kod üretme yeteneklerini karşılaştırdım. 

Yazılım mimarisi ve temiz kod prensipleri açısından değerlendirdiğimde, en başarılı sonuçları veren platformun **Cursor** olduğu kanaatine vardım. Ancak, genel kullanıcı deneyimi ve arayüz ergonomisi (workflow) açısından Cursor'ın kullanımının biraz hantal (clunky) hissettirdiğini gördüm. Bu nedenle daha pratik olan Antigravity ekosistemini ana çalışma ortamım olarak konumlandırdım.
